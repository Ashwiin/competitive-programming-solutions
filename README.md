## 🧭 How to Navigate

1. **By Platform**: Choose either `leetcode/` or `hackerrank/` directory
2. **By Topic**: Navigate to the specific topic folder (e.g., `arrays`, `strings`, `trees`)
3. **By Problem**: Each problem has its own folder containing:
   - `problem.md` - Problem description, constraints, and examples
   - Solution file(s) - Implementation in various languages

## 📝 Problem Format

Each problem folder follows this structure:

```
ProblemName/
├── problem.md          # Problem statement and description
└── ProblemName.java    # Solution implementation
```

### Problem Description (`problem.md`)
- Problem title and source link
- Problem statement
- Input/Output format
- Constraints
- Example test cases
- Approach and explanation
- Time and space complexity

### Solution File
- Clean, well-commented code
- Multiple approaches (if applicable)
- Edge case handling
